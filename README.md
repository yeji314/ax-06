# 🏠 부동산 추천 Agent

LangGraph 기반 State Machine으로 사용자의 조건(지역, 예산, 면적 등)을 분석하여 최적의 부동산 매물을 추천하는 AI Agent입니다.

---

## 🔄 Agent 구조 (Flow)

```
[START]
   │
   ▼
┌─────────────────────┐
│  parse_condition    │  ← 자연어 입력을 조건(JSON)으로 파싱 (LLM)
└─────────────────────┘
   │
   ▼
┌─────────────────────┐
│    validate         │  ← 지역 또는 거래유형 존재 여부 검증
└─────────────────────┘
   │
   ├─── is_valid=False & retry<2 ──▶ ┌─────────────────┐
   │                                  │    clarify      │  ← 부족한 조건 질문 (LLM)
   │                                  └─────────────────┘
   │                                         │
   │                                         └─── (루프) → parse_condition
   │
   ▼ (is_valid=True 또는 retry>=2)
┌─────────────────────┐
│ search_and_filter   │  ← 매물 검색 + 조건 필터링 + 점수 계산
└─────────────────────┘
   │
   ▼
┌─────────────────────┐
│    recommend        │  ← 자연어 추천 텍스트 생성 (LLM)
└─────────────────────┘
   │
   ▼
[END]
```

`.env.example`을 복사하여 `.env` 파일을 생성하고 OpenAI API 키를 입력합니다:

```

### 시나리오 1 — 정상 조건 입력

```
🏡 어떤 매물을 찾으시나요? 마포구에서 월세 보증금 3000에 월 80 이하 투룸 구해줘

📝 입력: 마포구에서 월세 보증금 3000에 월 80 이하 투룸 구해줘

파싱된 조건:
┌──────────────┬─────────┐
│ 항목         │ 값      │
├──────────────┼─────────┤
│ 지역         │ 마포구  │
│ 거래유형     │ 월세    │
│ 최대보증금   │ 3000    │
│ 최대월세     │ 80      │
│ 방종류       │ 투룸    │
└──────────────┴─────────┘

📊 1차 검색: 3개 → 필터링 후: 2개

✨ 추천 매물

🏆 1순위 추천: 마포구 공덕동 투룸 (P001)
- 보증금 3,000만원 / 월세 80만원
- 45m², 5호선 공덕역 도보 5분
- 장점: 역세권, 엘리베이터, 남향 채광 우수
- 추천 이유: 예산 조건 딱 맞고 편의시설 풍부

2순위: 마포구 신수동 투룸 (P011)
- 보증금 1,500만원 / 월세 70만원
- 38m², 6호선 광흥창역 도보 7분
- 장점: 한강 인근, 예산보다 저렴
```

## 🔧 사용 Tool 설명

### `search_properties` (tools/search_tool.py)
- **역할**: `data/mock_properties.json`에서 지역/거래유형/방종류 조건으로 1차 검색
- **입력**: `region`, `deal_type`, `property_type` (모두 선택)
- **출력**: 조건에 맞는 매물 목록

### `filter_and_score` (tools/filter_tool.py)
- **역할**: 검색 결과를 가격/면적 조건으로 2차 필터링하고 점수 계산 후 정렬
- **점수 기준**:
  - 가격 조건 충족: +30점
  - 면적 조건 충족: +20점
  - 역세권(도보 7분 이내): +15점
  - features 개수 × 5점
- **출력**: 점수 내림차순 상위 5개 매물
